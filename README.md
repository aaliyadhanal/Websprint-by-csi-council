# Websprint-by-csi-council
A responsive and user-friendly web portal for managing and showcasing campus events. This project allows users to view upcoming events, register for participation, and explore event details with an intuitive interface. Built using HTML and CSS, it emphasizes clean design, accessibility, and mobile responsiveness.

Features:

Event listing with date, time, and location details

Registration form for participants

Responsive design for mobile and desktop

Interactive UI elements for better user experience

Easy to customize and extend for any campus or organization

Technologies Used:

HTML5, CSS3

Responsive design using Flexbox and Media Queries

Use Case:
Ideal for colleges, universities, and student organizations to manage and display campus events efficiently.
